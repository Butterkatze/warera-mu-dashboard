# WarEra - Deutsche Military Units Dashboard

Ein minimalistisches React-Dashboard zur Anzeige und Verwaltung der offiziellen deutschen Military Units (MUs) für das Spiel WarEra. Das Tool liest die aktuellen Einstufungen automatisch aus einem In-Game-Artikel aus und zeigt sie auf einer Website an.

Dieses Projekt ist zu 99% Ai slop, aber war auch nur als kleine private Spielerei gedacht..
